How should my files be formatted?

1. File Type

Only Excel spreadsheets (.xlsx) are supported.

2. Header Row

The first row must contain column headers. Columns can be in any order.

3. Required Columns

Mandatory fields (e.g. External ID, Subsidiary) must not be empty or blank.

4. Valid Lists & Types

Dropdown list values (like Locations or Departments) must match approved NetSuite options.
